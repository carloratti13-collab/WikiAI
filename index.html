Technical Specifications

How It Works

VisioWikiAI uses state-of-the-art natural language processing to:

  1. Search Wikipedia for the requested topic
  2. Extract the relevant content
  3. Generate a concise summary using transformer-based AI models
  4. Display the results in a user-friendly format

AI Models

We use the following AI models:

  • DistilBART-CNN-12-6: A lightweight version of BART optimized for summarization tasks
  • Custom fine-tuned models: For specific domains and languages (coming soon)

Limitations

While VisioWikiAI is powerful, there are some limitations:

  • Complex technical topics may require additional context
  • Very long articles might exceed the model's input limits
  • Summaries are generated automatically and may occasionally miss nuances
  • Requires internet connection for Wikipedia searches

Code Examples

Core Functionality

Here's the core Python code that powers VisioWikiAI:

import wikipedia
from transformers import AutoModelForSeq2SeqLM, AutoTokenizer

def search_wikipedia(topic, language="en"):
    """Search for a topic on Wikipedia and return the page content."""
    wikipedia.set_lang(language)
    try:
        page = wikipedia.page(topic)
        return page.content
    except wikipedia.exceptions.PageError:
        return None
    except wikipedia.exceptions.DisambiguationError:
        return None

def summarize_text(text, model_name="sshleifer/distilbart-cnn-12-6", max_length=150):
    """Generate a summary of the given text using AI."""
    tokenizer = AutoTokenizer.from_pretrained(model_name)
    model = AutoModelForSeq2SeqLM.from_pretrained(model_name)

    inputs = tokenizer(text, max_length=1024, return_tensors="pt", truncation=True)
    summary_ids = model.generate(inputs["input_ids"], max_length=max_length, min_length=30, num_beams=4)

    summary = tokenizer.decode(summary_ids[0], skip_special_tokens=True)
    return " ".join(summary.split())

GUI Implementation

The graphical interface is built with PyQt5:

from PyQt5.QtWidgets import QApplication, QMainWindow, QVBoxLayout, QWidget, QLabel, QLineEdit, QPushButton, QTextEdit

class WikipediaSummarizerApp(QMainWindow):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("VisioWikiAI")
        self.setGeometry(100, 100, 800, 600)

        # UI setup code here
        self.init_ui()

    def init_ui(self):
        # Create widgets and layouts
        self.topic_input = QLineEdit()
        self.topic_input.setPlaceholderText("Enter a topic to search...")

        self.search_button = QPushButton("Generate Summary")
        self.search_button.clicked.connect(self.on_search_clicked)

        self.result_text = QTextEdit()
        self.result_text.setReadOnly(True)

        # Layout setup
        layout = QVBoxLayout()
        layout.addWidget(self.topic_input)
        layout.addWidget(self.search_button)
        layout.addWidget(self.result_text)

        container = QWidget()
        container.setLayout(layout)
        self.setCentralWidget(container)

Installation

To install and run VisioWikiAI:

  1. Clone the repository: git clone https://github.com/yourusername/VisioWikiAI.git
  2. Create a virtual environment: python -m venv venv
  3. Activate the environment:
    • Windows: venv\Scripts\activate
    • macOS/Linux: source venv/bin/activate
  4. Install dependencies: pip install -r requirements.txt
  5. Run the application: python wikipedia_summarizer_app.py
